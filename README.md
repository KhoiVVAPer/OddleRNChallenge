# OddleAssessmentProject

## Install
- Clone project.
- Move to challenge features branch.
 ```sh
  git checktout challenge
 ```
- install packages
  ```sh
  yarn 
  or
  npm i
  ```
- For run android:
  ```sh
  yarn android
  ```
- For run ios:
  ```sh
  cd ios && pod install && cd ..
  yarn ios
  ```
